# Portfólio Profissional — Gabriel Ferreira

Portfólio desenvolvido para apresentar minha trajetória profissional, minhas experiências e projetos nas áreas de suporte de TI, automação e desenvolvimento web.

O projeto reúne estudos de caso baseados em necessidades reais, demonstrando a identificação de problemas, o desenvolvimento das soluções, as tecnologias utilizadas e os resultados alcançados.

## Sobre o portfólio

O site foi criado com uma identidade visual própria, navegação responsiva e páginas dedicadas para apresentar:

- Perfil profissional
- Experiência em suporte de TI
- Projetos de desenvolvimento web
- Automações aplicadas ao ambiente corporativo
- Estudos de caso
- Currículo profissional
- Formas de contato

## Projetos apresentados

### Sistema de Gestão da EBD

Aplicação web desenvolvida para centralizar chamadas, alunos, professores, visitantes, ofertas, rankings, histórico e relatórios da Escola Bíblica Dominical.

### Automação de Limpeza Remota em Massa

Ferramenta em PowerShell criada para executar, acompanhar e registrar rotinas de limpeza de disco em vários computadores de forma centralizada.

### Gerador de Encaminhamento Field

Aplicação web criada para agilizar e padronizar a elaboração de textos técnicos destinados ao atendimento presencial.

### Automação de Chamados em Lote

Extensão para navegador desenvolvida para organizar e automatizar etapas repetitivas no processamento de chamados.

### Toolkit de Suporte Técnico

Aplicação desktop modular que reúne ferramentas utilizadas na rotina de suporte técnico em um único ambiente.
